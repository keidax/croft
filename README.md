# croft

A Crystal wrapper for Objective-C APIs.

Inspired by [hoop](https://github.com/0x73/hoop).

## Installation

1. Add the dependency to your `shard.yml`:

```yaml
dependencies:
  croft:
    github: keidax/croft
```

2. Run `shards install`
